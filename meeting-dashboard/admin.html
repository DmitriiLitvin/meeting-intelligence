Как добавить митинг в дашборд

  1. Заполни форму выше и нажми "Обработать"
  2. Скопируй полученный JSON
  3. Создай файл meetings/YYYY-MM-DD.json в репозитории
  4. Запусти python3 update_index.py — обновит index.json
  5. Запушь в GitHub — дашборд обновится автоматически
